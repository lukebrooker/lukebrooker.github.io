How do can we keep the aspect ratio of a video (or any other element) across different screen sizes?

  1. fitvids.js
  2. max-width: 100%
  3. max-height: 100%
  4. Zero height padding hack

Demo

@import "compass";

body {
  font-family: Helvetica, sans-serif;
  margin: rhythm(1) auto;
  max-width: rhythm(40);
}

%media--ratio {
  position: relative;
  padding-top: rhythm(1);
  height: 0;
  margin: 0 0 rhythm(1);
  overflow: hidden;
  iframe,
  object,
  video,
  .media--ratio__inner {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
  }
}

.media--ratio-16-9 {
  @extend %media--ratio;
  padding-bottom: 56.25%; // 16:9
}

.media--ratio-4-3 {
  @extend %media--ratio;
  padding-bottom: 75%; // 4:3
}
